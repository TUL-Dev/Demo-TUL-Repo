# Contributors

Thank you to everyone who has contributed to this Git practice repository!

## How to Add Yourself

1. Create a new branch: `git checkout -b add-yourname`
2. Add your name below in the format shown
3. Commit your changes: `git commit -m "Add [Your Name] to contributors"`
4. Push your branch: `git push origin add-yourname`
5. Create a pull request!
